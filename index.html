About

Mindset & Approach

I believe great software is invisible — it just works, reliably and efficiently.

I'm a software developer with a focus on .NET back-end systems and Azure cloud architectures. My goal is always to write code that is not just functional but maintainable and scalable — code that a future colleague (or my future self) will actually enjoy reading.

Beyond the enterprise stack, I'm genuinely excited about how Applied Artificial Intelligence is reshaping development workflows. I experiment with Machine Learning pipelines, train Convolutional Neural Networks for computer-vision tasks, and explore how intelligent tooling can amplify developer productivity.

I treat every project as an opportunity to learn: clean architecture, domain-driven design, and solid DevOps practices are principles I try to apply consistently, whether I'm building a REST API, orchestrating microservices on Azure, or prototyping a new ML model in Python.
